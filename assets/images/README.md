# Images Directory

This directory contains images used throughout the Goobercraft documentation site.

## Image Guidelines

When adding images:

1. **Optimize file sizes**: Compress images before uploading
2. **Use descriptive names**: `server-spawn-area.jpg` instead of `image1.jpg`
3. **Appropriate formats**:
   - PNG for screenshots and images with text
   - JPG for photos and complex images
   - SVG for logos and simple graphics
