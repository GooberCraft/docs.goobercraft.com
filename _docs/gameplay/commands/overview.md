## DataPacks

- Better Armour Stands
- Coordinates HUD

    You can enable/disable the Coordinates HUD by doing: `/trigger ch_toggle` in the chat box

## Plugins

- CrisTreeCapitator

    You can enable CTC with the command `/tc toggle`. This allows you to instantly chop down a tree _(and other wood blocks)_ up to 200 blocks at once. **Please use carefully!**
- EssentialsX

| Command | Description |
|---------|-------------|
| `/afk` | Let everyone know you're AFK. This will also trigger automatically after 5 minutes |
| `/clearinventory` | Clear your entire inventory |
| `/delhome` | Remove your home teleport |
| `/disposal` | Quick get rid of items in your inventory without dropping them on the ground |
| `/hat` | Wear the held item/block as a hat |
| `/helpop` | Send a message to online operators for assistance |
| `/home` | Teleport to your home |
| `/mail` | Use the `/mail` command in game for instructions |
| `/msg`, `/pm`, and `/whisper` | Privately message a player in the format of `/command Player Message`. You can reply to PMs by doing `/reply` or `/r` |
| `/near` | Discover players that are near you _(200 blocks radius)_ |
| `/nickname` | Change your in-game nickname |
| `/sethome` | Create your home teleport |
| `/spawn` | Teleport to the world spawn |
| `/suicide` | Instant Death |

- GravesX

    You can view your last 5 graves and their locations using the command `/graves`
- GSit

    You can do any of the following: `/sit`, `/lay`, `/bellyflop`, `/spin`, and `/crawl`
- mcMMO

| Command | Description |
|---------|-------------|
| `/mcnotify` | Toggle mcMMO notifications |
| `/mcrank` | Doing `/mcrank [player]` will show you the stats of another player |
| `/mcscoreboard` | Control the scoreboard on the right of your screen |
| `/mcstats` | Briefly shows all your stats on the right side of the screen |
| `/acrobatics` | View acrobatics level and information |
| `/alchemy` | View alchemy level and information |
| `/archery` | View archery level and information |
| `/axes` | View axes level and information |
| `/crossbows` | View crossbows level and information |
| `/excavation` | View excavation level and information |
| `/fishing` | View fishing level and information |
| `/herbalism` | View herbalism level and information |
| `/maces` | View maces level and information |
| `/mining` | View mining level and information |
| `/repair` | View repair level and information |
| `/swords` | View swords level and information |
| `/taming` | View taming level and information |
| `/tridents` | View tridents level and information |

- Shop

    You can find the [Shop plugin video guide](https://www.youtube.com/watch?v=6lDM1aAhgv0) on YouTube.

| Command | Description |
|---------|-------------|
| `/shop` | Open Shop Menu GUI |
| `/shop list` | View count of shops on the server + your own shop count |
| `/shop notify owner` | Toggle transaction notifications for your own shop(s) |
| `/shop notify stock` | Toggle stock notifications for your own shop(s) |
| `/shop notify user` | Toggle your transaction notifications with other shop(s) |

- SkinsRestorer

| Command | Description |
|---------|-------------|
| `/skin clear` | Reset your skin to your Mojang configured skin |
| `/skin random` | Set your skin to a random skin |
| `/skin search` | Search for available skins |
| `/skin set` | Set your skin to another player or local server skin |
| `/skin update` | Update your skin to your Mojang configured skin |
| `/skin url "<url>"` | Set your skin from a URL _(URLs Limited)_ |
| `/skins` | Open SkinsRestorer GUI Menu |

- StatusX

| Command | Description |
|---------|-------------|
| `/status clear` | Clear your status |
| `/status help` | View the help menu |
| `/status list [availabilities\|statuses]` | Get a list of statuses and availabilities |
| `/status nosleep` | Toggle prompting players to not sleep |
| `/status set <status> <availability>` | Set your status |
| `/status view <username` | View someone's status |
