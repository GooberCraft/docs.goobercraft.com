GooberCraft Season 2 is powered by a custom Forge modpack built to enhance your Minecraft adventures with new biomes, powerful machines, mystical creatures, and loads of decorative blocks. Whether you're here to explore, build, farm, fight dragons, or just vibe with friendsthis pack has something for you.

This page highlights some of the key mods that define the Season 2 experience and give GooberCraft its unique flavor.

### Featured Mods

These mods are suppose to be the star of the show adding new items, features, and experiences to Minecraft.

| Mod | Description |
|-----|-------------|
| Alex's Mobs | Adds dozens of real and fantasy-inspired mobs with unique behaviors. |
| Biomes O' Plenty | Biomes O' Plenty adds dozens of new biomes for richer world generation. |
| Craftopia | Adds 58 crops, 26 fruit trees, and over 250 foods to cook and eatperfect for players who love farming and variety in their meals. |
| Create | A mechanical engineering mod with gears, belts, and contraptions for automation. |
| Deeper And Darker | Adds eerie new dimensions, mobs, and gear for high-stakes exploration. |
| Dungeon Crawl | Generates large, multi-level dungeons filled with mobs and loot. |
| End's Phantasm | End's Phantasm overhauls the End with new biomes, mobs, and structures for a richer endgame experience. |
| End Remastered | Revamps The End by requiring 12 unique eyes found in various structures, turning portal activation into a true adventure. |
| Farmer's Delight | Enhances farming and cooking with new crops, meals, and kitchen tools. |
| Ice and Fire: Dragons | Introduces powerful dragons, mythical creatures, and epic gear. |
| Macaw's | A suite of decorative mods for building windows, doors, roofs, and more. |
| Tectonic | Improves world generation with realistic terrain and beautiful landscapes. |
| Sophisticated Backpacks | Sophisticated Backpacks adds fully upgradeable backpacks with automation, filtering, and customization to enhance your inventory management. |
| Supplementaries | Adds interactive redstone gadgets and quality-of-life building tools. |
| YUNG's Better X | A suite of mods that enhance vanilla structures like strongholds and dungeons. |

### Server-Side Mods

These are server-side mods that are installed to help with performance or give us extra features and tools.

| Mod | Description |
|-----|-------------|
| AI Improvements | Optimizes mob AI to reduce server lag without changing gameplay behavior. |
| Alternate Current | Replaces vanilla redstone with a more performant, consistent implementation. |
| BadOptimizations | Reverses unnecessary optimizations that harm compatibility or gameplay. |
| Better Chunk Loading | Improves how chunks load to reduce lag spikes and stuttering. |
| Better Mob Griefing Control | Gives fine control over which mobs can cause block damage. |
| Canary | A performance mod that enhances entity ticking, chunk loading, and more. |
| Chunk Sending | Optimizes chunk data transmission between server and client for smoother play. |
| Clumps | Groups XP orbs together to reduce lag and visual clutter. |
| Connectivity | Helps identify and fix mod compatibility or networking issues. |
| Dynamic View | Dynamically adjusts server view distance based on performance. |
| Fast Async World Save | Speeds up world saving to reduce lag during save events. |
| FastFurnace | Improves furnace processing performance by caching recipe lookups. |
| Fastload | Optimizes mod and game loading times on startup. |
| FastWorkbench | Caches crafting recipes to improve crafting table performance. |
| Fix GPU Memory Leak | Resolves known GPU memory leak issues with some modded setups. |
| FTB Ranks | Adds a flexible permission and rank system for multiplayer servers. |
| Get It Together, Drops! | Improves how dropped items are grouped and behave. |
| GriefLogger | Logs block changes from explosions and griefing for admin review. |
| Let Me Despawn | Allows mobs to properly despawn instead of sticking around forever. |
| ModernFix | Provides many under-the-hood performance improvements and memory fixes. |
| More Dragon Eggs | Makes the Ender Dragon drop an egg every time it's killed. |
| No Farmland Trample | Prevents farmland from being destroyed by players or mobs. |
| Server Core | Optimizes various server-side systems for better multiplayer performance. |
| Smooth Chunk Save | Reduces lag from chunk saving operations. |
| Structure Essentials | Adds tools and commands for managing custom structures. |
| WorldEdit | Powerful in-game editing tool for building and managing large structures. |
