### Server Configuration

#### Command: /gamerule

| Item | Value | Description |
|------|-------|-------------|
| `doFireTick` | `true` | Controls fire spread |
| `keepInventory` | `false` | Controls inventory on death |
| `maxEntityCramming` | `24` | How many entities can occupy a block before they start taking damage |
| `mobGriefing` | `true` | Controls of mobs can change blocks |
| `playerSleepingPercentage` | `20` | Controls the amount of players required to sleep through the night |
| `pvp` | `true` | Controls if pvp is enabled at the global level |
| `randomTickSpeed` | `3` | How often random events are "ticked" such as crop growth, grass spread, etc. |
| `snowAccumulationHeight` | `0` | How deep snow can accumulate on a single block |
| `spectatorsGenerateChunks` | `false` | If spectators can generate chunks |

#### File: server.properties

| Item | Value | Description |
|------|-------|-------------|
| `difficulty` | `hard` | Impacts various game elements |
| `simulation-distance` | `6` | Radius in which chunks "tick" around a player |
| `view-distance` | `8` | Radius of chunks sent to the Mineacrft client |
