## Quick Start Guide

This guide will get you playing on Goobercraft in just a few minutes!

## Step 1: Check Requirements

Make sure you have:

- Minecraft Java Edition (version X.XX.X) - *Update with actual version*
- A stable internet connection
- Read and agreed to our [Community Rules](/docs/community-rules/overview/)

## Step 2: Connect to the Server

1. Launch Minecraft
2. Click "Multiplayer"
3. Click "Add Server"
4. Enter the server details:
   - **Server Name**: Goobercraft
   - **Server Address**: `play.goobercraft.com` *(Update with actual address)*
5. Click "Done"
6. Double-click the server to join

See [How to Connect](/docs/server-info/connect/) for detailed instructions.

## Step 3: First Steps In-Game

Once you're connected:

1. **Read the Welcome Message**: Important server information will be displayed
2. **Visit Spawn**: Explore the spawn area and read any signs with important info
3. **Claim Your First Area**: Use the claim system to protect your build area
4. **Gather Resources**: Start your survival adventure!

### Essential Commands

- `/spawn` - Return to spawn
- `/sethome` - Set your home location
- `/home` - Teleport to your home
- `/help` - View all available commands
- `/rules` - Quick access to server rules

## Step 4: Join the Community

Connect with other players:

1. **Join Discord**: (Add Discord link) - Chat, get help, and stay updated
2. **Introduce Yourself**: Say hi in chat!
3. **Ask Questions**: Don't hesitate to ask for help
4. **Read Documentation**: Browse these docs to learn more

## Next Steps

Now that you're set up:

- Review [Community Rules](/docs/community-rules/overview/) thoroughly
- Check out [Guidelines](/docs/guidelines/overview/) for best practices
- Learn about [Server Features](/docs/server-info/features/)
- Start building your base!

## Getting Help

Need assistance?

- Ask in-game chat
- Check the [FAQ](/docs/server-info/faq/)
- Join Discord for support
- Contact staff members

## Tips for New Players

1. **Start Small**: Build a simple starter base before ambitious projects
2. **Explore Carefully**: The world is vast but remember your coordinates
3. **Respect Others**: Be courteous to other players and their builds
4. **Claim Your Land**: Protect your builds from the start
5. **Join Events**: Participate in community events to meet players

Welcome to Goobercraft - enjoy your adventure!
