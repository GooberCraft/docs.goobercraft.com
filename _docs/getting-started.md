1. Make sure you are on Minecraft version 1.21.1, Once on the Minecraft client, click the **Multiplayer** button

   ![Minecraft main menu showing the Multiplayer button](/assets/images/Mulitplayer.PNG)

1. On the Multiplayer screen, click the **Add Server** button

   ![Multiplayer screen with the Add Server button highlighted](/assets/images/add server.PNG)

1. On the Edit Server Info screen, In the 1st box enter a name you want to call your server in the _Server Name_ textbox. Under 2nd _Server Address_ textbox, paste your server's address (IP and port)

   ![Edit Server Info screen with Server Name and Server Address fields](/assets/images/server info.PNG)

1. Once you've completely filled out your server information, click **Done**

   ![Edit Server Info screen with the Done button highlighted](/assets/images/done.PNG)

1. Once you've successfully added your server into the Multiplayer page, you can then connect either by clicking on the right-pointing arrow-shaped button over the server icon, double-click on the server, or click the **Join Server** button while your server is selected.

   ![Multiplayer screen with the Join Server button and server listing](/assets/images/join server.PNG)
